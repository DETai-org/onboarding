# DETai Onboarding

Этот репозиторий — точка входа для новых участников команды.

## What lives here
- **Issues**: задачи для онбординга и пошаговые чек-листы (инструменты, workflow, основы CI/CD).
- **Sandbox (песочница)**: безопасное пространство для проб, экспериментов и тестирования кода без риска затронуть production-репозитории.

## How to use
1. Открой вкладку **Issues** и начни с задач, помеченных `onboarding` + `level:basic`.
2. Следуй чек-листу внутри каждого issue и оставляй заметки или вопросы в комментариях.
3. Используй `/sandbox` исключительно для обучения и экспериментов.
