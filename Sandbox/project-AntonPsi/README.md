# AntonPsi Onboarding Quest Bot

Учебная песочница для проекта внутри onboarding-репозитория.

Цель проекта: собрать маленького onboarding-проводника, который сначала работает
как CLI-приложение без внешних токенов, а затем получает опциональный Telegram
adapter.

План реализации ведется в ClickUp Epic:

- EPIC: AntonPsi Onboarding Quest Bot
- WP1: Скелет проекта и контракт данных квеста
- WP2: Движок квеста и локальное хранение прогресса
- WP3: CLI-интерфейс и демонстрационный сценарий прохождения
- WP4: Telegram adapter, тесты и финальная сборка учебного проекта

Эта папка создана как стартовая точка. Рабочий код будет добавляться отдельными
ветками и pull request по правилу: one Work Package = one branch = one PR.
