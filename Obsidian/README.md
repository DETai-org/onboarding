# Обсидиан (Obsidian) — Установка и настройка

## Описание

В этом разделе находятся все необходимые файлы и настройки для работы с **Obsidian** в рамках проекта **DETai**.

**Obsidian** — это локальная среда для работы с заметками, черновиками и подготовительными материалами, которая поддерживает синхронизацию через облако и работу в оффлайн-режиме.

### Содержимое папки:

- **.obsidian/** — скрытая папка, в которой хранятся настройки и плагины Obsidian.
  - Включает в себя **плагины** и их **настройки**, **темы оформления** и прочие локальные настройки среды.
  
- **README.md** — файл с инструкциями по установке и настройке Obsidian.
  
- **Заметки и структура** — папка, в которой будут храниться ваши рабочие материалы, заметки и проектные файлы.

## Для чего это нужно

1. **Локальное хранилище**: Все данные хранятся на вашем устройстве в папке **Tasks_Center**, что даёт вам полный контроль над информацией.
2. **Готовая структура**: Структура папок и плагины уже настроены для работы с материалами проекта DETai.
3. **Оффлайн-работа**: Весь процесс работы с заметками и черновиками доступен в оффлайн-режиме.

### Рекомендации

- Установите **Obsidian** и создайте новый Vault в папке **Tasks_Center** на вашем диске **D**.
- Папка **.obsidian** с настройками и плагинами будет автоматически использована для синхронизации и настроек.
- Используйте эту папку для хранения всех черновиков, заметок и материалов, связанных с проектом.

**DETai** — это экосистема, где инструменты, как **Obsidian**, поддерживают продуктивную работу с материалами и проектами.
