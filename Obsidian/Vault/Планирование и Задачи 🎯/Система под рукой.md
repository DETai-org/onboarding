## Приемы чтобы не выбрасывало


### 1. ClientAliveInterval
1. **Откройте конфиг** `/etc/ssh/sshd_config`:
    
    ```bash
    sudo nano /etc/ssh/sshd_config
    ```

2. **Перезагрузите SSH-сервис**:  
    Чтобы изменения вступили в силу, перезагрузите SSH-сервис:
    
    ```bash
    sudo systemctl restart sshd
    ```

---

## 2. ✅ Вот команда для подключения с поддержкой соединения:

Тебе просто нужно подключаться **не обычной командой**, а вот так:

```
ssh -o ServerAliveInterval=60 -o ServerAliveCountMax=5 admin-v-rossii@89.169.137.137
```

### 💬 Что она делает:
- `ServerAliveInterval=60` — клиент будет каждые **60 секунд** посылать keep-alive сигнал.
- `ServerAliveCountMax=5` — если **5 раз подряд не получен ответ**, соединение рвётся (то есть через 5 минут).
