
### 1. Как вставить шаблон вручную

- Нажми `Ctrl + P`
- Введи: 
    ```
    Templater: Replace templates in the active file
    ```

Или просто используй горячую клавишу.
___
### 2. Быстрый вызов шаблонов

- `Alt + E` (английская раскладка) — вызывает список шаблонов

Выбери нужный файл.  Например:

- `log_entry_template.md` — Логи дня
- `day_plan_template.md` — План дня
- `move_tasks_inject.md` — **переносит незавершённые задачи** из предыдущего дня в текущий план.

🪄 Рекомендуемый порядок:
1. Вызови `day_plan_template`
2. После генерации — снова нажми Alt + E → выбери `move_tasks_inject`
___
### 3. Открыть консоль логов (DevTools)

- `Ctrl + Shift + I` (Windows)

___
### 4. Синхронизация заголовков (File Title Updater)

После создания файла можно обновить связи между: `Filename` (kebab-style), И `title` во frontmatter, И первым заголовком `H1`. Для этого используется плагин **File Title Updater**.

**Способы запуска:**
`Ctrl + P` →  `File Title Updater: Sync titles using default source`

ИЛИ

**Alt + U**
___

### ==4. Как выделить текст жёлтым==

- Выдели нужный текст
- Нажми `Alt + W`
- Команда: **Включить/отключить тег подсветки для выделения**
